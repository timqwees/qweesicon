# QweesIcons

![QweesIcons](https://github.com/timqwees/qweesicon/blob/main/icons/logo/Cover.png?raw=true)

🎨 Красивые и минималистичные иконки для Visual Studio Code

## ✨ Почему QweesCore Icons?

🎯 Минималистичный дизайн - Чистые и современные иконки

🌈 Поддержка всех технологий - От React до Rust

📁 Умные папки - Специальные иконки для src, components, api и др.

⚡ Мгновенная установка - Один клик и готово

## 🚀 Установка

Откройте VS Code

Нажмите Ctrl+Shift+X (или Cmd+Shift+X на Mac)

Введите в поиске: QweesCore Icons

Нажмите Install

Перезагрузите VS Code

## 🔥 Что внутри?

### 📁 Папки

src/          components/     api/
utils/        assets/         tests/
docs/         config/         build/
### 📄 Файлы

Web: js, ts, vue, react, css, scss

Backend: py, java, go, rust, php

DevOps: docker, git, env

Документы: md, pdf, docx
